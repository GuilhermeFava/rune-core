# Rune Core Notice

The future `rune-core` package is intended to publish implementations of engineering methods that are grounded in public literature and benchmark validation.

Important boundaries:

- Public papers, equations, and benchmark methodologies are not the same thing as redistributing third-party code or restricted datasets.
- Any future public release must verify package licenses, data provenance, and redistribution rights independently.
- Hosted providers, tiles, geocoding services, weather APIs, and premium workflow integrations are intentionally outside the first open-source scope.

This preparation folder is therefore a packaging and scoping draft, not a final relicensing statement for the whole repository.
